# Changelog

## 1.0.0 (2026-04-01)

- Initial release
- Device synchronization from Overkiz cloud API
- Support for roller shutters, blinds, screens, awnings, lights, garage doors, gates
- Position control with slider (0-100)
- Favourite position ("my") command
- Automatic state polling every 5 minutes
- Support for Somfy Europe, Australia, and North America servers
- French and English documentation
